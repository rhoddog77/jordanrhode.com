I made this travel widget for my WDMD 302 class at the University of Wisconsin: Stevens Point. The main focus of this project was to get us to learn and use different object oriented programming design patterns.

The main design pattern that I used for this assignment was the Model View Controller(MVC) design pattern. One class handled the parsing of my XML data, one class handled the construction of my interface, and another class controlled and stitched the two together. We had two weeks to complete this project and I had a lot of fun with it.
